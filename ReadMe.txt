📊 Sales & Inventory Management Dashboard

📌 Project Overview
This project is a Sales & Inventory Management System built using PostgreSQL for database management and Power BI for visualization.
It helps track sales performance, monitor inventory levels, and analyze business trends effectively.


🛠️ Tools & Technologies Used

#PostgreSQL – For creating and managing the database.

#DBeaver – For SQL querying and database administration.

#Power BI – For creating interactive dashboards and visualizations.

#SQL – For data analysis (exploratory queries, EDA, etc.).
